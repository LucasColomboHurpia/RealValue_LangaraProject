# Welcome to RealValue - a Langara College project by team Quasars

## Technologies used
- React js
- Node js(Express)
- MongoDB
- Tomtom API
- Puppeteer

## Setup Development Environment
To setup your environment, please first run npm install in the 'client-side' and 'server' directories to install the neccessary dependencies required to run this project. Start up your mongoDB server on your local machine and create a new mongodb database locally called 'realvalu'.
To setup mongoDB server locally on mac OS you can reference [https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-os-x/](https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-os-x/) and on Windows [https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-windows/](https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-windows/).

## Available Scripts

In the project 'client-side' directory, you can run:

### `npm start` 
This script is used to start the react application in development mode.

In the project 'server' directory, you can run:

### `npm run dev`
This script is used to start the express application in development mode.
